PWSCUP データ匿名化・再識別コンテスト

匿名化部門（守り）

生データを匿名化し、有用性を保ちつつ個人を特定できないデータを生成します。

スコア = 有用性 U x 安全性 S

再識別部門（攻め）

匿名化されたデータに対して攻撃し、元データとの対応関係を復元します。

スコア = Precision x Recall x 難易度加重

総合スコア

総合ポイント = 50/匿名化順位 + 50/再識別順位。両部門1位なら100ポイント。
