# Task-018: submitコマンド

## 概要
匿名化/再識別アルゴリズムを提出するCLIコマンド

## 依存タスク
- task-013, task-015, task-017

## 成果物
- `src/pwscup/cli/submit.py`

## 詳細

### 使用方法
```bash
pwscup submit anonymize ./my_submission/
pwscup submit reidentify ./my_submission/
```

### 処理フロー
1. 提出物パッケージのバリデーション
   - algorithm.py の存在確認
   - metadata.json の読み込み・検証
   - requirements.txt のホワイトリスト照合
2. 提出制限チェック（1日の上限、フェーズ上限）
3. 提出をDBに記録（status: pending）
4. パイプラインオーケストレーターに評価を依頼
5. 結果をDBに保存
6. 結果サマリを表示

### 出力例
```
✓ 提出を受け付けました (submission_id: 42)
  部門: 匿名化
  チーム: team_alpha
  実行時間: 45.2秒
  有用性スコア: 0.782
  安全性スコア(暫定): 0.651
  匿名化スコア(暫定): 0.509
```

### エラーハンドリング
- フォーマットエラー: 詳細なエラーメッセージと修正方法を表示
- タイムアウト: 実行時間上限を表示
- メモリ超過: 制限値を表示
- ホワイトリスト違反: 許可ライブラリ一覧を表示

## 完了条件
- 正常な提出物が受け付けられ、評価結果が表示される
- 不正な提出物に対して適切なエラーメッセージが表示される
