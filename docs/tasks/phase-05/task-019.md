# Task-019: evaluateコマンド（ローカル評価）

## 概要
参加者が手元でアルゴリズムを評価できるローカル評価コマンド

## 依存タスク
- task-013, task-017

## 成果物
- `src/pwscup/cli/evaluate.py`

## 詳細

### 使用方法
```bash
# 匿名化の評価（サンプルデータ使用）
pwscup evaluate anonymize ./my_submission/ --data data/sample/

# 再識別の評価
pwscup evaluate reidentify ./my_submission/ --data data/sample/
```

### submitとの違い
- DBへの記録なし
- 提出制限なし
- サンプルデータに対して実行
- サンドボックスなし（ローカル直接実行）で高速
- 結果の詳細表示（指標別内訳）

### 出力例（匿名化）
