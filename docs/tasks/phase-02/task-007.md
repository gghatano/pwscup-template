# Task-007: 補助知識データ生成

## 概要
再識別部門の攻撃者に提供する補助知識（背景知識）データの生成

## 依存タスク
- task-006

## 成果物
- `scripts/generate_auxiliary.py` — 補助知識データ生成スクリプト
- `data/auxiliary/` — 生成された補助知識データ

## 詳細

### 補助知識データの定義
再識別攻撃者が「外部から入手できる情報」をシミュレートするデータ。

- 元データからサンプリングしたレコードの**準識別子のみ**を含む
- 機微属性は含まない（これが攻撃者が推定すべき対象）
- サンプリング率: 元データの30〜50%（パラメータで調整可能）

### 出力形式
```csv
original_id,age,gender,zipcode,occupation,education
42,35,M,150-0001,engineer,master
7,28,F,160-0023,nurse,bachelor
...
```

- `original_id` は元データのidに対応（これが正解マッピングの基準）
- 準識別子は元データと同一値

### 正解マッピング（運営専用）
- `data/auxiliary/ground_truth.json` — 匿名化データの行番号と元データidの対応
- 再識別評価時にのみ使用し、参加者には非開示

## 完了条件
- 補助知識データが生成できる
- サンプリング率を変えて難易度調整ができる
