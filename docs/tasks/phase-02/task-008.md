# Task-008: サンプルデータ作成・配布パッケージ

## 概要
参加者に配布するサンプルデータセットの作成

## 依存タスク
- task-006

## 成果物
- `data/sample/` — サンプルデータ一式
  - `sample_original.csv` — サンプル生データ（小規模）
  - `sample_schema.json` — スキーマ定義
  - `sample_auxiliary.csv` — サンプル補助知識データ
  - `hierarchies/` — 汎化階層ファイル

## 詳細

### サンプルデータの特徴
- レコード数: 1,000件（本番より小規模で手元テスト用）
- 本番データと同一のスキーマ・類似の分布
- 参加者がローカルで匿名化・再識別の動作確認に使用

### 配布形式
- `data/sample/` ディレクトリをそのまま参加者に配布
- Git管理対象とする（小規模のため問題なし）

## 完了条件
- サンプルデータが生成され、`data/sample/` に配置されている
- スキーマバリデーションに適合する
- ベースラインアルゴリズム（task-022, 023）で正常に動作する
