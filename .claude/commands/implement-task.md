Task $ARGUMENTS の実装を行ってください。

## 手順

1. `docs/tasks/` からタスクファイルを読み込む
   - タスク番号からファイルを特定: `docs/tasks/phase-*/task-$ARGUMENTS.md`
2. 依存タスクの成果物が存在するか確認する
   - 依存タスクの成果物ファイルが存在しない場合は、先に依存タスクの実装を提案する
3. タスクの「成果物」「詳細」に従って実装する
4. タスクの「完了条件」に従ってテストを書く
5. テストを実行して全件パスすることを確認する
6. `ruff check` と `mypy` でlint/型チェックを実行する

## ルール
- 仕様書 `docs/02_specification.md` の該当セクションも必ず参照すること
- 既存コードとの整合性を保つこと
- テストが通らない場合は修正してから完了とすること
